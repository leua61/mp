# 六色边框速度调整

- 将 `.rg` 六色边框动画时长从固定 `7.5s` 改为 CSS 变量 `--rg-dur`。
- 在 `js/utils.js` 的 `fit(e)` 中根据真正发光边框所在的中心 `.sq` 宽高估算周长：`perimeter = 2 * (sqWidth + sqHeight)`。
- 动画时长按周长同比例计算：`duration = 8.625 * perimeter / firstPerimeter`，首个正常题卡作为基准。
- `8.625s = 7.5s * 1.15`，即相对于 v4 当前速度整体慢 15%。
- 首次测得的中心边框周长作为基准，因此当前页面的基准卡从 7.5s 调整为 8.625s；更大边框自动更慢、更小边框自动更快，保持线速度接近一致。
- 缓存版本号已从 `anim1` 提升到 `anim2`，`boot.js` 引用提升到 `v=7`。
